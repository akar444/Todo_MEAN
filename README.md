# TodoApp using MEAN stack

### Installation

```
Install Node, npm and mongo if any of these are not installed.

1. Clone the project in a directory using command -

        git clone https://github.com/akar444/Todo_MEAN.git

2. After cloning the project navigate to directory Todo_MEAN
3. Run command
        "npm install"
   to install all node dependencies.
4. Run command
        "bower install"
   to install all client side libraries like angular,bootstrap and jquery
5. Run command
           "npm run build"
6. Open another terminal and run mongo server "mongod".
7. Now run command
           "npm start"
      to start the server.It will start server at
           http://localhost:8000
      Open this in browser.
```
